# Calculadora em PyScript

Este projeto é uma calculadora simples, com tema do Super Mario, desenvolvida em **PyScript**, que permite realizar as quatro operações matemáticas básicas diretamente no navegador:

- Soma  
- Subtração  
- Multiplicação  
- Divisão  

## Estrutura do projeto

- `calculadora.py` → Código principal em Python (PyScript) com as funções de cada operação.  
- `index.html` → Página HTML que contém os campos de entrada (`num1`, `num2`) e o espaço para mostrar o resultado (`resultado`).  
- `README.md` → Documentação do projeto.  

## Exemplo de uso

1. Abra o terminal dentro da pasta do projeto.
2. Inicie um servidor local rodando o comando: `python -m http.server`
3. Abra o seu navegador e acesse o endereço: `http://localhost:8000`
4. Digite dois números nos campos de entrada.
5. Clique no botão da operação desejada.
6. O resultado aparecerá na área de saída e o som será reproduzido.
